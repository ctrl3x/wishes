# ЖЕЛАЙ

Компания представляет из себя салон искушений, где люди разыгрывают не деньги, а время жизни, память и свободу на исполнение своих желаний и учатся формулировать их так, чтобы цена была осознанной.
Желания действительно исполняются, но каждое имеет стоимость и последствия.
Здесь соблазн всегда изящен, а цена — скрыта в мелком шрифте.

ЖЕЛАЙ СМЕЛО, ИГРАЙ ОСОЗНАННО
